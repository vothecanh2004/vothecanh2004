### Mục tiêu tuần 7:

* Nắm vững cơ chế xác thực (Authentication) và phân quyền (Authorization) của AWS Cognito thông qua luồng User Pool và Identity Pool.
* Hiểu sâu về cấu trúc cấu tạo và cách thức hoạt động của JSON Web Token (JWT) trong bảo mật hệ thống.
* Khởi tạo, cấu hình hoàn chỉnh các chính sách bảo mật và luồng Sign Up/Sign In trên Cognito User Pool.
* Thực hành tích hợp thành công Cognito vào Frontend, xử lý luồng nhận JWT và gửi yêu cầu xác thực an toàn qua AWS API Gateway.

### Kết quả đạt được tuần 7:

* Hiểu rõ cơ chế xác thực (Authentication) và phân quyền (Authorization) của AWS Cognito .
* Nắm vững cấu trúc và cách hoạt động của JSON Web Token (JWT) trong việc bảo mật hệ thống.
* Khởi tạo và cấu hình thành công AWS Cognito User Pool với các chính sách bảo mật, mật khẩu và luồng Sign Up/Sign In.
* Tích hợp thành công Cognito vào Frontend, xử lý mượt mà luồng Đăng ký và Đăng nhập.
* Sử dụng thành công JWT Token để gửi yêu cầu và xác thực qua AWS API Gateway an toàn.
