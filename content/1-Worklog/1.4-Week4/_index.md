### Week 4 Objectives:

* Understand Amazon S3 object storage mechanisms and how to select cost-optimized Storage Classes based on access frequency.
* Gain foundational knowledge of relational database service Amazon RDS (SQL) and non-relational database service Amazon DynamoDB (NoSQL).
* Gain hands-on proficiency in: creating S3 Buckets, configuring Object access permissions, and provisioning and connecting to a MySQL Instance on Amazon RDS.

### Week 4 Achievements:

* Mastered core knowledge of Amazon Simple Storage Service (Amazon S3).
