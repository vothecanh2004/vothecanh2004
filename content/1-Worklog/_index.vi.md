Trang Nhật ký công việc này tổng hợp lộ trình thực tập kéo dài 12 tuần của tôi, phản ánh tiến độ hoàn thành từ các mục tiêu lý thuyết đến sản phẩm thực tế.


Lộ trình được cấu trúc bài bản qua từng tuần với nội dung như sau:

**Tuần 1:** [Làm quen với AWS, Cloud Computing, tạo tài khoản AWS và sử dụng AWS Management Console.](1.1-week1/)

**Tuần 2:** [Tìm hiểu IAM & Security, quản lý người dùng, phân quyền và các nguyên tắc bảo mật trên AWS.](1.2-week2/)

**Tuần 3:** [Tìm hiểu Amazon EC2, triển khai và quản lý máy chủ ảo trên AWS.](1.3-week3/)

**Tuần 4:** [Tìm hiểu Amazon S3, Amazon RDS và Amazon DynamoDB, thực hành lưu trữ dữ liệu và quản lý cơ sở dữ liệu.](1.4-week4/)

**Tuần 5:** [Tìm hiểu Amazon VPC, CloudWatch và CloudTrail, xây dựng hạ tầng mạng và giám sát hệ thống.](1.5-week5/)

**Tuần 6:** [Khởi tạo dự án Quizizz Clone, thiết kế cơ sở dữ liệu, giao diện và thiết lập hạ tầng AWS ban đầu.](1.6-week6/)

**Tuần 7:** [Tìm hiểu và tích hợp Amazon Cognito để xây dựng chức năng xác thực người dùng.](1.7-week7/)

**Tuần 8:** [Xây dựng Backend REST API, phát triển các chức năng quản lý Quiz, Question và tích hợp Amazon S3, Amazon RDS.](1.8-week8/)

**Tuần 9:** [Xây dựng các chức năng Realtime bằng API Gateway WebSocket và tích hợp Amazon SQS để xử lý nộp bài hàng loạt.](1.9-week9/)

**Tuần 10:** [Thực hiện kiểm thử API bằng Postman, kiểm tra và hoàn thiện các chức năng của hệ thống.](1.10-week10/)

**Tuần 11:** [Tối ưu hiệu năng AWS Lambda, giám sát hoạt động bằng Amazon CloudWatch.](1.11-week11/)

**Tuần 12:** [Tích hợp toàn bộ hệ thống, kiểm thử End-to-End.](1.12-week12/)
