### Week 5 Key Achievements:

* Mastered core knowledge of Amazon Virtual Private Cloud (Amazon VPC), understanding how AWS establishes an isolated virtual network environment and provisions IP address allocations (CIDR blocks) to manage system infrastructure.
* Differentiated clearly between Public Subnets and Private Subnets; grasped network routing architectures utilizing Internet Gateways (for Public Subnets) and NAT Gateways (enabling secure, one-way outbound internet connectivity for Private Subnets).
* Gained a comprehensive understanding of AWS's primary monitoring and auditing tools:
  * Amazon CloudWatch: Tracked performance metrics, monitored resource utilization, and configured automated threshold Alarms.
  * AWS CloudTrail: Logged historical activities, operations, and system-wide API invocations inside the AWS account to maintain robust security auditing and compliance standards.
* Consolidated and systematized the full spectrum of core AWS concepts explored throughout the training journey, successfully bridging key service tracks together: Identity Management (IAM), Compute (EC2), Storage (S3), Databases (RDS, DynamoDB), Networking (VPC), and Monitoring (CloudWatch, CloudTrail).b
