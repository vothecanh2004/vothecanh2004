### Week 1 Objectives:

* Understand Cloud Computing fundamentals: basic concepts, service models (IaaS, PaaS, SaaS), and deployment models (Public, Private, Hybrid Cloud).
* Master AWS Global Infrastructure (Region, AZ, Edge Location, Local Zone) and the Shared Responsibility Model.
* Create an AWS Free Tier account and gain proficiency in navigating the AWS Management Console.
* Configure AWS Budgets to manage costs and set up billing alerts, ensuring usage stays within Free Tier limits.
